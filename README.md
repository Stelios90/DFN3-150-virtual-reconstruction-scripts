# DFN3-150-virtual-reconstruction-scripts

DFN3-150 virtual reconstrutions is the GitHub repository accompanying the following publication:

Koutalis, S., Röding, C., Merceron, G., Guy, F., Kostopoulos, D.S., & Harvati, K. (2026). Virtual reconstruction and analysis of the face of DFN3-150 Paradolichopithecus aff. arvernensis specimen from Dafnero, Greece. Scientific Reports 

This publication, as well as the underlying groundwork carried out by Dr. Stefan Schlager (see references in the method section of the named publication), should be credited in case of use. The use should be research-oriented and non-commercial.
